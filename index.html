Symmetry Blue

Here at Teams we use the Symmetry Blue mobile app to scan various card readers at key access doors throughout our various locations.

To Get Setup

  1. Download the mobile app from the Apple Store or Google Play Store
  2. Look for the icon with the silhouette of an eagle
  3. Ensure to allow access to Bluetooth and location services
  4. Once the application is installed, expand the menu option in the top and select "view device ID"
  5. The IT department will need the device ID to grant you access

Entrapass Web

Download/Setup

  1. Get Entrapass from: EntraPass web
  2. Click download and follow the instructions on screen
  3. Install the app
  4. Copy the path from the URL: http://64.4.78.100:81/EntrapassWeb/

Update Device ID

  1. Go to Card Holder > Search Users
  2. Select user
  3. Ensure device number is card number 1
  4. Under "Options" you can change the pin to zero and the last 4 digits of the user's SIN (Eg: 08765)

Email Management

Managing Storage / Mass Deleting Emails

Important: Before deleting emails, sort and organize your inbox. It is HIGHLY RECOMMENDED to sort your inbox into folders to separate important emails.
  1. Visit outlook.com and login
  2. Go to Settings > Account > Storage
  3. Identify the folder you wish to mass delete emails from
  4. Select the drop down arrow beside "Empty"
  5. Choose deletion options (All emails, 3 months and older, etc.)
  6. Keep the window open until the operation is complete

Imaging

  1. From Azure remote desktop > Start > All apps > Trans Plus Systems folder > Open User Manager Pro > Login
  2. Make Folder: Go to \\teamsdc2019\FMImages > Create user folder following naming scheme
  3. Create desktop shortcut for the folder on user's computer
  4. In Fleet > Options > Document Imaging Setup
  5. Right click under "Storage Definitions" > Add Storage Location
  6. Follow naming convention for "Storage Definition Name"
  7. Set "Store indexed Documents in" to \\teamsdc2019\FMImages\Storage
  8. Set "Documents to index will be" to \\teamsdc2019\FMImages\[User]'s Imaging Queue
  9. Click Test > Save
  10. Back in "Document Imaging" window, select user name and right click in "Related Document Definitions" area > Add Document Definitions
  11. Enter "Username's Imaging Queue" as document name
  12. Select "Probil" from dropdown
  13. Go to Security tab > Select "All users" > Save

Biz Connect Mobile Application

Setup

  1. Navigate to your app store and search "Biz Connect"
  2. Install the application (icon with telephone company logo)
  3. Login with your credentials
  4. If unable to login, select "Forgot Password" and follow instructions

Change Availability and Forward Calls

  1. Select your profile icon in the top left
  2. Hover over "Available" to see options:
    • Available: Calls come to your phone
    • Do Not Disturb: Calls go to voice mail
    • Invisible: Status hidden, calls work normally
  3. To forward calls, toggle on "Forward Calls"
  4. Select where forwarded calls should go (coworker, call queue, external number)
  5. Optionally specify start and end dates for automated forwarding

Teams Audio Setup

Speaker Issues - No Sound

  1. Check volume using speaker icon on taskbar
  2. Try F3 (increase volume) or F2 (decrease volume) on Dell keyboards
  3. Check Sound Settings via search bar
  4. Ensure correct output device is selected
  5. Check settings on Local User Account if using cloud
  6. During call: Select More > Audio > Adjust volume
  7. Check Sound Mixer settings for MS Teams app

Microphone Issues

  1. Search "Sound Mixer" and ensure correct input device is selected (typically "Echo Speaker Phone")
  2. In Sound Mixer, locate MS Teams and ensure correct input device is selected

Teams Work From Home Policy

  1. Staff must have documented permission from their manager to work from home
  2. Specifics regarding date and time must be clearly outlined with all concerned parties
  3. WFH staff must maintain high communication levels (video/audio calls via PC and cell)
  4. Must have a private and secure work environment
  5. Minimum 150 Mbps upload and download internet speed
  6. Must submit to Teams MDM (mobile device management) policy

Personal Machine Requirements

  • 16 GB RAM
  • i5 multi-core processor or 1.3 GHz minimum
  • 250 GB HDD
  • Web Cam
  • Speakers and Mic or Headset
